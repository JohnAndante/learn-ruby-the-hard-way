# The Truth Terms

* <code>and</code>
* <code>or</code>
* <code>not</code>
* <code>!= => not equal</code>
* <code>== => equal</code>
* <code>>= => greater than equal</code>
* <code><= => less than equal</code>
* <code>true</code>
* <code>false</code>

<br>
